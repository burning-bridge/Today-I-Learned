## MD 문법

* **HEAD LINING** `#` : 문서의 제목과 부제목을 나타낼 때. #의 갯수로 중요도 변경됨

* **DIVIDER** `--- or /// or <hr>`: 줄로 나눠줌

* **글머리** : 순서가 필요하다면 숫자. 아니면 * , + 또는 -으로

* **코드 블럭** 

  1. 코드박스 ` ``` `: 언어선택 가능
  2. 인라인코드` `` `: 짧은 코드/ 코드 일부만 쓸 때 사용

* **강조**

  1. **굵게**`**텍스트**`
  2. *이탤릭* `*텍스트*`
  3. ***굵은 이탤릭*** `***텍스트***`
  4. ~~취소선~~`~~텍스트~~`

* 표는 |를 이용해 만들 수 있음 ex) `| | |` ->2줄 3칸의 표로

  ! 무조건 2줄 이상으로 생성!

* 인용 `>` : >>를 쓰면 인용 인덴트는 2칸
